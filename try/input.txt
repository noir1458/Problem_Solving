3. 시간 복잡도 (Time Complexity)
행렬 체인 곱셈 문제의 상향식 접근법의 시간 복잡도는 O(n^3)입니다. 이는 다음과 같이 계산됩니다:

외부 루프는 체인의 길이 l을 2에서 n까지 증가시키며 n번 반복합니다.
중간 루프는 시작 인덱스 i를 1에서 n-l+1까지 증가시키며 n번 반복합니다.
내부 루프는 k를 i에서 j-1까지 증가시키며 n번 반복합니다.
따라서 총 연산 횟수는 O(n^3)이 됩니다.

4. 설명
초기화: 대각선 요소 m[i][i]는 0으로 초기화됩니다. 이는 한 개의 행렬을 곱하는 데 비용이 없음을 의미합니다.
이중 루프: 첫 번째 루프는 체인의 길이 l을 증가시키며, 두 번째 루프는 시작 인덱스 i를 증가시키며 순차적으로 계산합니다.
최소 곱셈 비용 계산: m[i][j]는 행렬 Ai에서 Aj까지 곱하는 데 필요한 최소 곱셈 횟수를 저장합니다. 내부 루프에서 k를 기준으로 나누어 계산 비용을 최소화합니다.
결과 저장: 최종적으로 m[1][n]은 전체 행렬 체인을 곱하는 데 필요한 최소 곱셈 횟수를 나타냅니다.
최적 괄호 묶기 출력: s 배열을 사용하여 최적의 괄호 묶는 방법을 재구성할 수 있습니다.
